**CLK** : Clock pin's at PA5 (For STM)

**SDA** : I2C Data Pin

**VDD** : 3.3V for Stm32

**SA0** : ID pin's of Sensor. 

**GND** : Ground Pin. 

**GND2** : Another Ground Pin for ID.
